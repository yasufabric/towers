# 戦国タワーディフェンス 〜天下統一〜

戦国時代を舞台にしたブラウザ向けタワーディフェンスゲームです。  
江戸幕府の守護者として、押し寄せる敵軍を塔で撃退し、天下統一を目指しましょう。

🎮 **[プレイする](https://yasufabric.github.io/towers/)**

---

## 概要

- 敵が通るルート沿いのマスに塔を配置して攻撃する、タワーディフェンス型のゲームです
- 20ウェーブを耐え抜いたらクリアです
- 城の耐久度が0になるとゲームオーバーです

| ステージ | 舞台 | 難易度 |

## 塔の種類

| 塔 | コスト | 特徴 |
|---|---|---|
| 🏹 弓兵塔 | 50文 | 基本攻撃 |
| 🥷 忍者塔 | 100文 | 速射・毒 |
| 💣 砲撃塔 | 150文 | 範囲爆発 |
| ⚔️ 侍塔 | 200文 | 高威力貫通 |
| ☯️ 陰陽師 | 175文 | 減速魔法 |

- 塔は3段階までアップグレード可能
- 売却すると購入額の50%が返還されます

## 敵の種類

足軽・騎馬武者・鬼・天狗・武者・忍・火鬼・大将（ボス）  
ウェーブが進むにつれて強力な敵が登場します。

## 攻略のコツ

- 道の曲がり角付近に塔を配置すると射程を有効に活用できます
- 陰陽師で敵を減速させてから他の塔で集中攻撃するのが効果的です
- ウェーブ開始前に戦略を立てて塔を配置しましょう
